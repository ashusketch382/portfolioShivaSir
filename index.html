Projects



Design and analysis of speed control of DC motor using microcontroller

In this project speed is controlled of a DC motor using microcontroller which is integrated with chopper. By using pulse width modulation technique chopper is controlled which inturn controls the output voltage which is proportional to the speed of the motor

Powerfactor Measurement by using DAQ6009

In this project, powerfactor is measured by using data acquisition card(DAQ6009),in which the supply is fed to RLC-loads in which current is measured by current sensor by placing a current transformer in series with load ,a potential transformer is placed parallel to sense the voltage across them. By using LABVIEW software phases of both current and voltage are determined and subtracted whose cosine value gives power factor.

Design, Analysis and Implementation of DC-DC converters

In this project, DC-DC converters like boost, forward, flyback converters etc., are designed, analysed, tested. Designing of magnetic elements like inductor, transformer and non-magnetic elements like capacitor, power electronic switch like mosfet (IRFZ44) is done. Mathematical analysis like getting stable frequency response by using bode plot is done and practically the circuit is connected. The analog IC TL494 is used to get required pulses and the required ouput DC has been obtained.

Development of Soft Switching Synchronous Buck Converter Topologies for Low Voltage High Current Applications

In this, new topologies have been discovered for low voltage high current applications. These soft switching converters reduce switching losses which is a major concern in the DC-DC converters. This in turn enhances the efficiency and also alleviates the voltage and current stresses.
